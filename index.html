一、git的基本语法命令 git init test (git创建名为test的仓库) cd test (进入test仓库) git status (查看仓库状态) git add . (把所有文件添加到暂存区) git commit -m "描述" (提交版本并且备注说明) git log (查看版本记录) git checkout id (回滚到那条记录，之后的记录全没，id是这条记录id至少前七位) git log --oneline (换行) git tag -a v1 -m "描述" (tag标签 -a为标签名 -m为描述) git tag (结束标签) x code (苹果下才可以用) q (退出查看版本记录) k (往上) j (往下) 二、git提示 no commits yet (还没有任何记录) untracked files (未跟踪到的文件) modified (已更改)
